# guang_yi
基于爬虫的翻译程序

<br/>

## 描述
#### 项目名称: 光译
#### 主要功能: 1.词典(查单词) 2.翻译(查句子) 3.生词本(记单词)
#### 环境: python3 mysql5.5 PyCharm PyQt5

<br/>

## 环境搭建:  
  先准备好数据库, sql脚本我已经放在根目录, 然后直接运行项目中的py文件即可
